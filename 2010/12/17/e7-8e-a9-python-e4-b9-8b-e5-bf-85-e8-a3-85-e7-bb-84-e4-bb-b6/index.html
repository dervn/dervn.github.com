玩 Python 之必装组件
via Numb3r3 by Charlie on 12/16/10

搞python的web开发，必然需要安装的一些组件。记录一下我在拿到一台新机器之后必装的 Python 组件。

  • simplejson http://pypi.python.org/pypi/simplejson Imaging (PIL) http://www.pythonware.com/products/pil/ Tornado http://www.tornadoweb.org/ MySQLdb http://sourceforge.net/projects/mysql-python/ couchdb-python http://code.google.com/p/couchdb-python/ redis-py http://github.com/andymccurdy/redis-py Twisted http://twistedmatrix.com/ Storm https://storm.canonical.com/ SQLAlchemy http://www.sqlalchemy.org/ paramiko 模块，用来模拟ssh登录的 http://www.lag.net/paramiko/ Get/Post 利器 pycurl http://pycurl.sourceforge.net/

当然，还有 CouchDBX，redis (via MacPorts)，memcached (via MacPorts) 和 GAE SDK。
